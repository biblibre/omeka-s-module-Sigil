Configuration

Pour accéder à la page de configuration, connectez-vous à l’interface d’administration, allez sur la page « Modules », trouvez « Sigil » et cliquez sur le bouton « Configurer » correspondant.

Pour sélectionner l’image à utiliser comme filigrane, cliquez sur le bouton « Choisir ». Cela ouvrir la barre latérale des fichiers, où vous pourrez charger un nouveau fichier, ou sélectionner un fichier existant.

Pour plus d’informations à propos des fichiers, voir Assets dans le manuel utilisateur Omeka S. (anglais)

Une fois l’image sélectionnée, vous pouvez choisir une position pour le filigrane. Les options disponibles sont :

  • Centre

  • Est

  • Nord Est

  • Nord

  • Nord Ouest

  • Sud Est

  • Sud

  • Sud Ouest

  • Ouest

Pour confirmer les changements de configuration, cliquez sur le bouton « Soumettre ».
