Sigil documentation
===================

`Sigil`_ is a module for Omeka S that automatically modify imported images to
add a watermark.

.. _Sigil: https://github.com/biblibre/omeka-s-module-Sigil

Requirements
------------

* Omeka S >= 4.0.0
* `ImageMagick <https://imagemagick.org/>`__

Quick start
-----------

1. `Add the module to Omeka S
   <https://omeka.org/s/docs/user-manual/modules/#adding-modules-to-omeka-s>`__
2. In the module configuration form, upload an image file that will serve as
   watermark and select a position for the watermark.
